# Reviews
  # CoDepencies
  Node.js, MonogoDb and npm are all required to run this app
  node and npm-> https://docs.npmjs.com/downloading-and-installing-node-js-and-npm
  mongoDB -> https://docs.mongodb.com/manual/mongo/
  # Getting Started
  1 As usual (npm install)
  2 (npm run db:setup)
  3 npm run dev:react
  4 npm start
